# nginx-for-ruby-on-rails-on-kubernetes
nginx for Ruby on Rails on kubernetes.

## Usage

```bash
$ kubectl create -f nginx-config.yaml
$ kubectl create -f nginx-for-ruby-on-rails-on-kubernetes-svc.yaml
$ kubectl create -f nginx-for-ruby-on-rails-on-kubernetes.yaml
```
